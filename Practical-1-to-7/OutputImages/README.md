# Output Images

This folder contains sample mock screenshots for Practical 1 to 7. 

### Instructions to Replace with Real Screenshots:
1. Capture your actual screenshots from the emulator.
2. Save them as PNG files named `Practical-1.png`, `Practical-2.png`, etc.
3. Replace the existing mock screenshots in this directory with your real screenshots.

Feel free to delete this README file if it’s no longer needed.
